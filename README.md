# Simone Dante Visconti - Portfolio

Portfolio personale sviluppato con React + Vite per presentare i miei progetti, competenze e identità come web developer.

🔗 Live: https://tuo-link-portfolio.com

---

## 🚀 Struttura
- ⚛️ React
- ⚡ Vite
- 🎨 CSS custom + Bootstrap
- 🎞 Framer Motion (animazioni)
- 🌗 Dark / Light Mode (custom)
- 🧠 Context API (gestione tema)
- 📦 React Icons

---

## ✨ Features
- Layout moderno e responsive
- Animazioni fluide (Framer Motion)
- Effetto typing nella Hero
- Dark mode persistente (localStorage)
- Sezione progetti dinamica
- Cookie widget personalizzato
- Sidebar con navigazione interna
- UI minimale e orientata UX
